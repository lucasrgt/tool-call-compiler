Spec viva do tool-call-compiler: compiler agnóstico que transforma múltiplas tool calls dependentes em um único feedback composto para a LLM. A versão atual registra IR/optimizer/runtime, MCP stdio persistente, adapters HTTP/fs/shell, limites por tool, cache plugável, JSON Schema do plano, SDK TypeScript e gates de 500 LOC produtivas + cobertura mínima de 95%.
